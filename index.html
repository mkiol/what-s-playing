What's Playing

An Add-on for Mozilla Firefox to display desktop notifications for Spotify and other music websites

Download this project as a .zip file Download this project as a tar.gz file

Installing the Add-on

Visit the Firefox Add-ons website on Firefox to get the Add-on.

Usage

Play a song on Spotify Web Player or Songza and see desktop notifications when a new song starts. You can also find out what song is playing at the moment by clicking on "What's Playing Now" from the context menu (right-click in a Firefox page).

Not Seeing Notifications?

If you are on a Mac, make sure notifications are enabled and Do Not Disturb is not on (especially if you are using an external display).

Contribute or Request a Feature

Fork What's Playing on GitHub. If you want to see a new feature or support for another music website or find the Add-on not working, please contact me by email or on Twitter.
